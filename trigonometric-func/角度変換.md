# 角度変換

## 90° − θ（余角）の公式

| 関数         | 公式       |
| ------------ | ---------- |
| sin(90° − θ) | = cosθ     |
| cos(90° − θ) | = sinθ     |
| tan(90° − θ) | = 1 / tanθ |

## 90° + θ の公式

| 関数         | 公式        |
| ------------ | ----------- |
| sin(90° + θ) | = cosθ      |
| cos(90° + θ) | = −sinθ     |
| tan(90° + θ) | = −1 / tanθ |

## 180° − θ（補角）の公式

| 関数          | 公式    |
| ------------- | ------- |
| sin(180° − θ) | = sinθ  |
| cos(180° − θ) | = −cosθ |
| tan(180° − θ) | = −tanθ |

## 180° + θ の公式

| 関数          | 公式    |
| ------------- | ------- |
| sin(180° + θ) | = −sinθ |
| cos(180° + θ) | = −cosθ |
| tan(180° + θ) | = tanθ  |

## 360° + θ の公式

| 関数          | 公式   |
| ------------- | ------ |
| sin(360° + θ) | = sinθ |
| cos(360° + θ) | = cosθ |
| tan(360° + θ) | = tanθ |

## −θ（負角）の公式

| 関数    | 公式    |
| ------- | ------- |
| sin(−θ) | = −sinθ |
| cos(−θ) | = cosθ  |
| tan(−θ) | = −tanθ |
